# Ejemplo sencillísimo de cómo usar JPA, Hibernate y Derby

## Tecnologías usadas

El proyecto tiene una estructura de projecto maven, ha sido editado usando eclipse.
El objetivo es usar JPA para guardar en base de datos de una aplicacion. La implementación
de JPA será Hibernate y la base de datos a utilizar será Derby.
